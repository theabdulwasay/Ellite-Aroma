**Ellite Aroma** is an elegant, fully responsive e-commerce storefront dedicated to showcasing luxury fragrances, traditional Ittars, and premium signature perfumes. Designed with a cohesive, high-end visual identity centering around rich gold typography and deep emerald tones, it delivers a flawless boutique experience directly in the browser.

## ✨ Key Features

* **👑 Immersive Luxury UI:** Cohesive emerald-and-gold color palettes echoing premium perfume houses.
* **📱 Ultra-Responsive Layout:** Perfected using modern CSS Flexbox and Grid to guarantee a smooth experience on smartphones, tablets, and wide screens.
* **🛍️ High-Conversion Cards:** Interactive product display cards with hover transitions and clear calls to action.
* **📂 Organized Collections:** Structured categorizations separating authentic, traditional Ittars from modern spray perfumes.
* **⚡ Clean Semantic HTML:** Compliant and accessible structures optimizing both browser performance and SEO.

---

## 🛠️ Built With

* **HTML5:** Structural layout and semantic markup.
* **CSS3:** Custom page stylings, flexible grids, and polished transitions.
* **Google Fonts:** Upscale font combinations to complement a luxury aesthetic.
