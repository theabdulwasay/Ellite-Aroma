## Introduction
The goal of **Ellite Aroma** is to provide a digital storefront that reflects the elegance of its products. Built with **HTML5** and **CSS3**, the website features a responsive layout, interactive elements, and a cohesive visual identity centered around Gold and Emerald themes.

## Features
- **Responsive Design:** Fully adaptable layout for desktops, tablets, and mobile devices.
- **Interactive UI:** Smooth hover effects, transitions, and engaging product cards.
- **Navigation:** easy-to-use navigation bar for quick access to collections and information.
- **Product Showcases:** Dedicated pages for both Ittar and Perfume collections with detailed product grids.
- **Semantic HTML:** clean and accessible code structure.

## Technologies Used
- **HTML5:** For structural foundation and semantic content.
- **CSS3:** For styling, including Flexbox, Grid, and responsive media queries.
- **Google Fonts:** Utilized for premium typography.



## Installation & Usage

   ```bash

   ```

   ```bash
   cd ellite-aroma
   ```
3. **Open the project:**
   Simply open `index.html` in your preferred web browser (Chrome, Firefox, Edge, etc.) to view the website.
